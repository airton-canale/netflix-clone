## Installation

- Important: You must have installed the React.js and Git

```bash
# clone it
$ git clone git@github.com:airton-canale/netflix-clone.git

$ cd netflix-clone

# Install dependencies
$ npm install

# Running scripts
$ npm start
```
- And enter in your browser using http://localhost:3000
